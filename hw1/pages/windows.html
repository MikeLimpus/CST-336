Windows

Microsoft Windows is the most common desktop operating system.[1] Windows is a closed source operating system. It was originally created as an addon program to MS-DOS,[2], but evolved and eventually replaced DOS as Microsoft's primary operating system. Being the most popular operating system means that Windows enjoys the greatest level of software compatibility, with the vast majority of commercial software and games being compatible with Windows.
